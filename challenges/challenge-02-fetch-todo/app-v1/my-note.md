توش با ریکت هوک اشنا تر شدم و یکم دستم باهاش گرم شد

فچ و ارور هندلینگ هم دستم براش گرم شد

راستش من کلا از ریکت هوک استفاده نمیکردم. همیشه یه از هلپر های فانکشنی استفاده میکردم. ولی برای این جور چیزا که از فیچر های دیفالت ریکت مثل استیت و اینا استفاده میکنه بدک نیست کلا سوییچ کنم روش

و ترکیب کردن اطلاعات لوکال استورج با داده های فچ شده جالب بود

فکر کنم داره حسابی ... برام جا میوفته. برای ایتریت کردن توی ارری استفاده میشه
